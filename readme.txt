== Description ==
This plugin provides demo content import functionality for the MindBridge WordPress theme.
It works with the One Click Demo Import plugin to import demo content, widgets, and customizer settings.

== Installation ==
1. Install and activate the One Click Demo Import plugin.
2. Upload and activate the MindBridge Demo Importer plugin.
3. Activate the MindBridge theme.
4. Navigate to Appearance → Import Demo Data.
5. Click Import.

== Notes ==
This plugin is intended to be used only with the MindBridge theme.
